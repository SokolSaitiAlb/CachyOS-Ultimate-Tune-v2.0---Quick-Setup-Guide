## ✨ What's New in v2.3

### 1. Enhanced Limine Bootloader Configuration ⭐

**Advanced timeout and entry selection:**

```bash
# /boot/limine.conf

# Timeout: Seconds before automatic boot
timeout: 5

# Default Entry: Which entry to boot by default
default_entry: 2
```

**Features:**
- ⏱️ Configurable timeout (in seconds)
- 🎯 Direct entry selection
- 📁 Directory entry support with autoboot tips
- 🔧 Full configuration examples

**Important Tip:**
- If `default_entry` points to directory (`/+CachyOS`), autoboot is disabled
- To autoboot entry within directory, point to specific entry number
- Entries numbered starting from 1

**Example Configuration:**
```conf
timeout: 5
default_entry: 2

/+CachyOS              # Entry 1: Directory
//linux-cachyos        # Entry 2: Bootable kernel
    protocol: linux
    kernel_path: boot():/vmlinuz-linux-cachyos
    cmdline: quiet splash root=UUID=... rw
    module_path: boot():/initramfs-linux-cachyos.img
```

### Basic Setup (Default):
```bash

default_entry: 1
```

### Boot CachyOS by Default:
```bash
timeout: 5
default_entry: 2  # linux-cachyos entry
```

```bash

default_entry: 1
```

### Extended Timeout (10 seconds):
```bash
timeout: 10
default_entry: 2
```

### Edit Configuration:
```bash

# Make changes
# Save (Ctrl+O, Enter, Ctrl+X)
```

## 🖼️ SDDM Theme Setup

### Install via Script:
- Select "Install SDDM Pixel UI Theme" in menu
- Script handles installation automatically
- Restarts SDDM after installation

### Manual Installation:
```bash
# Clone theme
git clone -b pixel https://github.com/mahaveergurjar/sddm.git

# Install
sudo mv sddm /usr/share/sddm/themes/

# Configure
sudo nano /etc/sddm.conf
# Set: [Theme] Current=sddm

# Restart
sudo systemctl restart sddm
```

### Preview Theme:
```bash
sddm-greeter-qt6 --test-mode --theme /usr/share/sddm/themes/sddm
```
