## ⚙️ v2.3 New Features

### Limine Bootloader Configuration (Enhanced)
**Timeout Settings:**
- `timeout: 5` - Seconds before automatic boot (configurable)
- `default_entry: 1` - Which entry to boot by default
- Advanced entry selection
- Directory support with tips

**Edit Configuration:**
```bash
sudo nano /boot/limine.conf

# Example:
timeout: 5
default_entry: 2  # Boot linux-cachyos by default
```
